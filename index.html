HTML

  • Hypertext Markup Language (HTML) gives the content structure.
  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • The img element contains information about images that are displayed on the webpage.
  • The src element attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The p element represents a paragraph or block of text.

CSS

  • Cascading Style Sheets(CSS)gives the content style
  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS style sheet, and External CSS style sheet.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page,
  • and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code
  • External CSS style sheet allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • git branch list your branches. a * will appear next to the currently active branch
  • git branch [branch-name] create a new branch at the current commit
  • git checkout switch to another branch and check it out into your working directory
  • git merge [branch] merge the specified branch’s history into the current one
  • git status show modified files in working directory, staged for your next commit
  • git add [file] add a file as it looks now to your next commit (stage)
  • git push [alias] [branch] Transmit local branch commits to the remote repository branch
  • git pull fetch and merge any commits from the tracking remote branch
  • git fetch [alias] fetch down all the branches from that Git remote
  • git add -A The git add command adds modifications in the current working branch to the staging area.The "- A" flag indicates that we want to add all changes
  • git commit -m
